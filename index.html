# Start With V2ray GUI

Welcome to the V2ray GUI client tutorial. This tutorial will guide you to configure your v2rayN, Qv2ray, v2rayNG, Kitsunebi and Netch client.

Before starting, please make sure that you already have a V2ray server.If you don’t have it, please go to Google Play to download V2vpnapp.You can also build your own V2ray server by following V2ray guide.
